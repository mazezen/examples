# Flak Quick start

Flask: https://flask.palletsprojects.com/

## Create an environment
```shell
$ mkdir flask.palletsprojects.com-quickstart
$ cd flask.palletsprojects.com-quickstart
$ python3 -m venv .venv
```

## Active the environment
```shell
$ . .venv/bin/activate
```
